# 参与者公约

## 我们的承诺

为建设开放、友好、多元的环境，我们承诺：参与我们社区的所有成员都不分年龄、体型、身体健全与否、民族、性征、性别认同与表达、经验水平、教育程度、社会地位、国籍、相貌、种族、宗教信仰或性取向，均不应受到骚扰。

## 我们的准则

有助于创造积极环境的行为包括：

- 使用友好、包容的语言
- 尊重不同的观点与经验
- 耐心接受建设性批评
- 关注对社区最有利的事情
- 对其他社区成员保持同理心

参与者不可接受的行为包括：

- 使用与性有关的言语或图像，以及不受欢迎的性关注或性企图
- 挑衅、侮辱/贬损的评论、人身攻击或政治攻击
- 公开或私下的骚扰
- 未经明确许可发布他人的私人信息（如物理地址、电子地址）
- 其他在专业场合中合理认为不恰当的行为

## 我们的责任

项目维护者有责任阐明可接受行为的标准，并对任何不可接受行为采取恰当、公平的纠正措施。

项目维护者有权利和责任删除、编辑或拒绝与本行为准则不符的评论、提交、代码、wiki 编辑、Issue 及其他贡献，并可暂时或永久禁止任何他们认为不适当、威胁、冒犯或有害行为的参与者。

## 适用范围

本行为准则适用于所有项目空间，也适用于个人在公共空间代表本项目或其社区时（例如在官方社交媒体、官方活动、线上/线下活动中的代表身份）。

## 执行

如遇辱骂、骚扰或其他不可接受行为，可通过 GitHub Issue（非安全类）或维护者可见渠道联系项目团队。维护团队将审查并调查全部投诉，并据情形做出被认为必要且适当的回应。维护者有义务对报告者保密。具体执行政策可能另行发布。

未切实遵守或执行本行为准则的项目维护者，可能被项目领导层其他成员临时或永久地终止其参与。

## 来源

本行为准则改编自 [Contributor Covenant](https://www.contributor-covenant.org/zh-cn/version/2/1/code_of_conduct/) 2.1 版。
